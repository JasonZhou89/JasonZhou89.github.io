前言：推荐编辑器：vscode

markdown vscode插件

  • markdown all in one:快捷键+数学公式
    • Ctrl + B 粗体
    • Ctrl + I 斜体
    • Alt + S 删除线
    • Ctrl + Shift + ] 标题(uplevel)
    • Ctrl + Shift + [ 标题(downlevel)
  • markdown preview enhanced：绘制流程图、插入 LaTeX 公式、运行代码（如果要运行自己写的语句的话，需要手动打开设置：Enable Script Execution）
    • MPE 可以简单地把其他文件导入到当前文档中，语法是：
      • @import “你的文件”
