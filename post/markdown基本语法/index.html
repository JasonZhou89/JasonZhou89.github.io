三、Markdown列表

Markdown支持有序列表和无序列表。

3.1 无序列表使用*、+、-作为列表标记：

  • 第一项
  • 第二项
  • 第三项
  • 第四项
  • 第五项
  • 第六项

3.2 有序列表使用数字加.号表示：

  1. 第一项
  2. 第二项
  3. 第三项

3.3 列表嵌套

列表嵌套只需要在子列表中选项前加4个空格即可：
1. 第一项
* 第一项 第一元素
* 第一项 第二元素
2. 第二项 + 第二项 第一元素 - 第二项 第二元素 * 第2项第二元素下第一子项
